**Driving License Application**

**Description**:
It is a desktop application made to run a driving licesnse system.

**Features:**
> It has a Login system.
> 
> You can add people and not every person is a user.
> 
> You can add users or use people to make them users.
> 
> It has 7 types of applications ( Local Driving License, International License, Renew License,...etc).
> 
> There are tests system should be done befor issuing the Licnese.
> 
> You can add drivers.
> 
> You can control your user setting (Password, username, ..etc).
> 
> All features has validations.

**Used Technologies:**
> C# (C sharp)
> 
> .Net Framework
> 
> Windows Forms (Win Forms)
> 
> SQLClient (SQl server Database)
> 

**Important to Do After Clone:**
. You should run the SQL script then go to "properties" of the database created then "Files" then you find "DVLD.mdf", move to its location
 copy it then paste it in "DVLD\DVLD\Database".
 
 Note: The name of it should be exactly same as I wrote.
